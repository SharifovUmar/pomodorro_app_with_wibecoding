## Приложение для отслеживания времени работы Pomodoro

### Описание
Приложение для отслеживания времени работы Pomodoro с использованием PyQt5. В приложении реализованы функции таймера Pomodoro, настройки, уведомления и возможность скрыть окно приложения в трей.

### Важно
Код сгенерирован нейронной моделью и является vibe coding-ом.
